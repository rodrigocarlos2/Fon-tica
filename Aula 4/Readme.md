
O Encontro Consonantal define a ocorrência de consoantes uma ao lado da
outra. Não se pode confundir com Dígrafo, Dígrafo é um grupo de letras
(que podem ser consoantes) que representam um único som. O Encontro 
Consonantal é um grupo de consoantes (exclusivamente), em que cada
uma realiza emissão de som.

Os tipos de Encontro Consonantal variam conforme a divisão silábica do
mesmo. Se o Encontro Consonantal permanecer na mesma sílaba, então, 
chama-se de Encontro Consonantal Perfeito.

Caso o Encontro Consonantal se divida em sílabas diferentes, chama-se,
então, de Encontro Consonantal Imperfeito. No caso de uma palavra ter
ocorrência de divisão perfeita e imperfeita, denomina-se como Encontro
Consonantal Misto. Uma curiosidade é que o Encontro Consonantal Misto
sempre possui no mínimo um encontro de três consoantes.
